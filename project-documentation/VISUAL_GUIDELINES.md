# Visual Guidelines

**Last Updated:** 2025-11-24

---

## Color Palette

**Primary Brand:** Purple gradient (#667eea to #764ba2)
**Usage:** Primary CTAs, brand elements, hero sections

**Neutrals:** Deep navy to white
**Usage:** Text, backgrounds, surfaces

**Accent:** Purple-pink gradient
**Usage:** Highlights, hover states, visual interest

---

## Layout Patterns

### Hero Section
- Deep purple gradient overlay
- Large typography (8xl on desktop)
- Translucent glass cards
- Centered content with max-width 7xl

### Content Sections
- Alternating backgrounds (white/muted gray)
- Generous padding (py-20 on desktop)
- Max-width containers (7xl)
- Card-based layouts with glass morphism

### Grid Systems
- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3-4 columns
- Gap: 4-8 spacing units

---

## Visual Style

**Glass Morphism:**
- Translucent backgrounds (95% opacity)
- Backdrop blur (12px)
- Subtle borders (30% opacity)
- Soft shadows

**Typography:**
- Gobold for display headlines (bold, wide tracking)
- Inter for body text (clean, readable)
- Progressive scaling (mobile to desktop)

**Animations:**
- Subtle hover effects (scale 105%)
- Fade-in-up on scroll
- Text shimmer for hero headlines
- Respect reduced motion preferences

---

## Component Preferences

- Glass cards over solid backgrounds
- Gradient buttons for primary CTAs
- Icon + text for navigation items
- Expandable cards for content sections
- Sticky navigation on scroll
